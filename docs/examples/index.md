# Examples

Twelve runnable demos. Each page has a `.ts` sibling you can copy into your own project.

| # | Title | What it shows |
| - | ----- | ------------- |
| [01](./01-basic-map.md) | Basic map | OSM basemap + marker + popup. |
| [02](./02-camera.md) | Camera | Bearing, pitch, `flyTo`, `easeTo`. |
| [03](./03-vector-tile.md) | Vector tiles | Synthetic MVT through `VectorTileMapLayer`. |
| [04](./04-style-spec.md) | Style spec | `setStyle` + dynamic `setPaintProperty`. |
| [05](./05-heatmap.md) | Heatmap | 500 random points with an animated gradient. |
| [06](./06-hillshade.md) | Hillshade | Synthetic DEM + hillshade preview. |
| [07](./07-clusters.md) | Clusters | 5,000 points in `GeoJSONClusterSource`. |
| [08](./08-symbols.md) | Symbols | Labeled POIs with collision. |
| [09](./09-geocoder.md) | Geocoder | Search box wired to Nominatim. |
| [10](./10-directions.md) | Directions | Click two points, draw OSRM route. |
| [11](./11-offline.md) | Offline | Download a bbox, toggle the network. |
| [12](./12-globe.md) | Globe | Canvas2D orthographic projection preview. |
